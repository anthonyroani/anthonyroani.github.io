    Link a skill to an Employee

    Description

    An Employee can have one or more skills. This page describes the way to link or unlink an existing Skill to an Employee.

    Info

    As we have a many-to-many relationship between Skill and Employee, we have created a intermediate table that we used to match skills with employees.

    Relationships : read, linking and delete

    Read the skills of an Employee

    GET

    https://cvteque.herokuapp.com/employees/:id/skills
    EXAMPLE USAGE
    https://cvteque.herokuapp.com/employees/99/skills

    PARAMETERS

    Field Type Description
    id Int The employee's ID you want to see.
    Response example : an array of skills
    {
  "skills": [
    {
      "name": "C#",
      "skill_details": {
        "skillLevel": 4
      },
      "id": 4,
      "color": "#BBBBBB"
    },
    {
      "name": "PHP",
      "skill_details": {
        "skillLevel": 4
      },
      "id": 6,
      "color": "#BBBBBB"
    }
  ]
}

    Link a skill for an Employee

    Warning

    Skills must already exist in the database before linking a Skill to an Employee.

    This endpoint links a Skill to an Employee.

    POST

    https://cvteque.herokuapp.com/employees/:id/skills
    EXAMPLE USAGE
    https://cvteque.herokuapp.com/employees/99/skills

    PARAMETERS

    Field Type Description
    id Int The employee's ID you want to link a Skill.
    Requirements

    When linking a Skill to an Employee, it needs to know the skill level of the Skill.

    Just post it with the following json.

    EXAMPLE OF JSON
      {
    "skillLevel":4
  }

    Delete a specific Experience

    DELETE

    https://cvteque.herokuapp.com/employees/:id/skills/:id
    Usage example
    https://cvteque.herokuapp.com/employees/99/skills/3

    Parameters

    Field Type Description
    id Int The employee's ID you want to delete.
