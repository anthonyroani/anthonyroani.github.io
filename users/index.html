    Users

    Details

    User is the Entity that is responsible to manage all the Users that are registered in the app, it regroups some generic properties such as identity, e-mail, etc.

    A User have a One to One relationship with Employee as an Employee is a sub-entity of User.

    Properties

    Field Type Description
    firstName String The first name of the user
    lastName String The last name of the user
    compagnyName String optional The compagny name where is working the user
    ldap Unique String optional The LDAP identifier of the user if exists
    email Unique String The e-mail address of the user
    employeeId Int optional The foreign key ID of the associated Employee

    Info

    All parameters required by routes of the API need to be passed as JSON format

    Read all users

    GET

    https://cvteque.herokuapp.com/users

    Read a specific user

    GET

    https://cvteque.herokuapp.com/users/:id
    EXAMPLE USAGE
    https://cvteque.herokuapp.com/users/99

    PARAMETERS

    Field Type Description
    id Int The user's ID you want to see.

    Create a user

    POST

    https://cvteque.herokuapp.com/users
    EXAMPLE OF JSON
    {
  "created object": {
    "lastName": "Roani",
    "compagnyName": "Wordline",
    "firstName": "Anthony",
    "email": "anthonyroani@gmail.com",
    "ldap": "ARI",
    "employeeId": null
  },
  "success": true
}

    Update a user

    PATCH

    https://cvteque.herokuapp.com/users/:id
    EXAMPLE USAGE
    https://cvteque.herokuapp.com/users/99

    PARAMETERS

    Field Type Description
    id Int The user's ID you want to update.
    EXAMPLE OF JSON
    {
    "firstName":"Anthony",
    "lastName": "Roani",
    "compagnyName":"Genesis groupe",
    "ldap":"",
    "email":"a.roani@genesis-groupe.com"
}

    Delete a specific user

    https://cvteque.herokuapp.com/users/:id
    EXAMPLE USAGE
    https://cvteque.herokuapp.com/users/99

    PARAMETERS

    Field Type Description
    id Int The user's ID you want to delete.
