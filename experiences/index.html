    Experience

    Details

    Experience is the class that is responsible to manage all the experiences associated to a Collaborator.

    Properties

    Field Type Description
    trigram String The trigram of the collaborator (example: ARI)
    availableDate String The date when the collaborator ends its mission. Format is DD-mm-yyyy.

    CRUD Operations

    All parameters required by routes of the API need to be passed as JSON format

    Read all experiences

    GET

    https://cvteque.herokuapp.com/experiences

    Read a specific experience

    GET

    https://cvteque.herokuapp.com/experiences/:id
    EXAMPLE USAGE
    https://cvteque.herokuapp.com/experiences/99

    PARAMETERS

    Field Type Description
    id Int The experience's ID you want to see.

    Create a collaborator

    POST

    https://cvteque.herokuapp.com/experiences
    EXAMPLE OF JSON
    {
    "trigram":"ARI",
    "availableDate": "12-06-2017"
}

    Update an experience

    PATCH

    https://cvteque.herokuapp.com/experiences/:id
    EXAMPLE USAGE
    https://cvteque.herokuapp.com/experiences/99

    PARAMETERS

    Field Type Description
    id Int The experience's ID you want to update.
    EXAMPLE OF JSON
    {
    "trigram":"DLI",
    "availableDate": "18-09-2018"
}

    Delete a specific experience

    DELETE

    https://cvteque.herokuapp.com/experiences/:id
    EXAMPLE USAGE
    https://cvteque.herokuapp.com/experiences/99

    PARAMETERS

    Field Type Description
    id Int The experience's ID you want to delete.
